🐶🐱 Dog-vs-Cat-Image-Classification
A Deep Learning project that classifies images as Dog or Cat using Convolutional Neural Networks (CNN).
This project was developed and trained using Google Colab.

📌 Project Overview

This project builds an image classification model that can accurately distinguish between dog and cat images. It uses a CNN architecture trained on labeled image data and evaluated using standard performance metrics.

The notebook includes:
- Data preprocessing
- Model building
- Training & validation
- Performance evaluation
- Prediction on new images

🛠️ Tech Stack

- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- Google Colab

⚙️ Model Architecture
- Convolutional Layers
- Max Pooling Layers
- Flatten Layer
- Fully Connected (Dense) Layers
- Output Layer (Binary Classification)

Activation Functions:
- ReLU (Hidden Layers)
- Sigmoid (Output Layer)
- Loss Function:
- Binary Crossentropy

Optimizer:
- Adam

🚀 How to Run the Project
- Open the notebook in Google Colab
- Upload the dataset (Dogs vs Cats dataset)
- Run all cells sequentially
- Train the model
- Evaluate performance
- Test predictions on new images
